## 简介

uView Pro 是全面支持 Vue 3.0 + TypeScript 的 uni-app 生态 UI 框架，基于 uView 1.8.8 使用 TypeScript 完全重构，兼容多端（微信小程序、支付宝小程序、H5、App、鸿蒙等）。

## 特性

- 🚀 **Vue 3 + TypeScript** - 现代化技术栈，完整的类型支持
- 📱 **多端兼容** - 支持微信小程序、支付宝、百度、字节跳动、QQ 小程序、H5、iOS/Android App、鸿蒙
- 🧩 **80+ 精选组件** - 功能丰富，开箱即用
- 🎨 **多主题定制** - 支持深色模式、自定义主题
- 🌍 **国际化 i18n** - 内置多语言支持
- 📦 **按需引入** - 精简打包体积
